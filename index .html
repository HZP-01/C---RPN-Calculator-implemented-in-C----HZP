RPN Calculator 使用说明书

一个功能丰富的逆波兰表示法计算器，基于C++实现，支持基本运算、高级数学函数和完整的栈操作。

1. 安装和编译指南

系统要求

  • Windows 10 或更高版本
  • C++编译器 (MinGW/g++)
  • 支持C++11标准

项目文件结构

echo "项目文件:"
echo "├── rpn_calc.cpp         # 主程序文件"
echo "├── rpn_calculator.h     # 头文件"
echo "└── rpn_calculator.cpp   # 实现文件"
项目文件:
├── rpn_calc.cpp         # 主程序文件
├── rpn_calculator.h     # 头文件
└── rpn_calculator.cpp   # 实现文件

echo "# 1. 使用MinGW编译所有文件"

3. 所有支持的操作说明

基本算术运算

| 操作符 | 描述 | 语法 | 栈操作 |

|--------|------|------|---------|

| + | 加法 | a b + → a+b | 弹出2个操作数，压入1个结果 |

| - | 减法 | a b - → a-b | 弹出2个操作数，压入1个结果 |

| * | 乘法 | a b * → a*b | 弹出2个操作数，压入1个结果 |

| / | 除法 | a b / → a/b | 弹出2个操作数，压入1个结果 |

高级数学函数

| 命令 | 描述 | 功能 | 示例 |

|------|------|------|-------|

| fib n | 斐波那契数列 | 计算第n个斐波那契数 | 10 fib → 55 |

| pascal n | 杨辉三角 | 计算杨辉三角第n行的和 | 5 pascal → 32 |

栈操作命令

| 命令 | 描述 | 功能 | 示例输出 |

|------|------|------|----------|

| clear | 清空栈 | 移除栈中所有元素 | clear → 栈已清空 |

| stack | 显示栈 | 显示当前栈中所有元素 | stack → 栈: 5 10 15 |

系统命令

| 命令 | 描述 | 功能 |

|------|------|------|

| help / h | 显示帮助信息 | 查看所有可用命令和示例 |

| q / quit / exit | 退出程序 | 安全退出计算器 |

4. 错误代码和异常说明

运行时错误

| 错误类型 | 错误信息 | 原因 | 解决方法 |

|----------|----------|------|----------|

| 栈为空 | 错误: 栈为空 | 尝试从空栈弹出元素 | 确保栈中有元素再进行操作 |

| 操作数不足 | 错误: 操作符 'X' 需要两个操作数 | 二元操作符缺少足够的操作数 | 检查栈中是否有足够元素 |

| 除零错误 | 错误: 除零错误 | 除数为0 | 确保除数不为0 |

| 无效参数 | 错误: fib需要参数 / pascal需要参数 | 高级函数缺少参数 | 在函数前提供数字参数 |

| 无效输入 | 错误: 斐波那契数列需要非负整数 | 参数为负数或非整数 | 使用非负整数参数 |

| 未知token | 错误: 未知token 'X' | 输入了无法识别的命令 | 检查拼写，使用help查看可用命令 |

| 空表达式 | 错误: 空表达式 | 没有输入任何内容 | 输入有效的表达式 |

| 无结果 | 错误: 没有结果 | 表达式未产生结果 | 确保表达式包含有效的计算 |

输入验证规则

  • 数字格式: 支持整数和小数（例: 5, 3.14, -2.5）
  • 操作符验证: 检查是否为支持的操作符
  • 栈状态检查: 操作前验证栈是否满足要求
  • 参数验证: 高级函数验证参数有效性
